# AngularStaticBlog

Angular + Scullyで作る静的ブログアプリ

## Create Markdown File

```shell script
$ ng g @scullyio/init:post --name="<File Name>" --target="blog"
```

## Local build & serve

```shell script
$ npm run scully:serve
```

## Dev Tools & Stack
- ESLint
- Prettier
- Jest
- Angular Material(Custom Color)
